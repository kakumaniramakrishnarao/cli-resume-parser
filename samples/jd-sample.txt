**Job Title:** Full-Stack AI Engineer
**Location:** Remote (U.S. or India preferred)
**Company:** NovaScale Technologies

---

## About Us

NovaScale Technologies is building intelligent SaaS platforms that combine modern full-stack engineering with cutting-edge AI capabilities. Our mission is to help businesses automate workflows, extract insights from unstructured data, and build smarter applications.

---

## Role Overview

We are looking for a Full-Stack AI Engineer who can design, build, and deploy scalable web applications integrated with AI/ML systems. You will work across the stack—from frontend interfaces to backend APIs and AI pipelines—delivering production-grade features.

---

## Responsibilities

### Full-Stack Development

* Build and maintain web applications using modern frameworks (React, Next.js)
* Develop backend services and APIs using Node.js, Python, or similar
* Design and optimize database schemas (PostgreSQL, MongoDB)
* Ensure performance, scalability, and security of applications

### AI / Machine Learning Integration

* Integrate LLM APIs (e.g., OpenAI, Gemini) into applications
* Build prompt engineering workflows and structured output pipelines
* Develop data processing pipelines for unstructured data (PDFs, text, etc.)
* Implement vector search systems (FAISS, Pinecone, Weaviate)

### System Design & DevOps

* Architect scalable systems for AI-powered applications
* Deploy services using Docker, Kubernetes, and cloud platforms (AWS/GCP/Azure)
* Implement CI/CD pipelines and monitoring
